# Random dungeon generator

Random dungeon generator from https://news.ycombinator.com/item?id=19309378, reformatted and deobfuscated.

## Building with Make

```
make dungeon
```

Then you can run the compiled executable

```
./dungeon
```

## Building with CMake

```
mkdir cmake-build-debug
cd !$
cmake ..
make
```

Then you can run the compiled executable

```
./dungeon
```
