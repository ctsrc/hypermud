# ASCII Random Dungeon Generator

Random dungeon generator from https://news.ycombinator.com/item?id=19309378, deobfuscated, refactored and commented.

## Building with Make

Create a binary that produces deterministic dungeons (for testing / development).

```
CFLAGS=-DDETERMINISTIC_DUNGEONS make dungeon
```

Create a binary that produces random dungeons.

```
touch dungeons.c
make dungeon
```

Run the compiled executable

```
./dungeon
```

## Building with CMake

```
mkdir cmake-build-debug
cd !$
cmake ..
make
```

Run the compiled executable

```
./dungeon
```
